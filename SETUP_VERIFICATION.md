## ğŸ¨ Customization Quick Tips

### Change Colors
Edit `css/style.css`, find `:root` section:
```css
:root {
    --primary-color: #2563eb;      /* Change this */
    --secondary-color: #1e40af;
    /* ... other colors ... */
}
```

### Change Election Name
Edit `data/mock-data.json`:
```json
"election": {
    "name": "Your Election Name"
}
```

### Add Polling Station
Edit `data/mock-data.json`, add to `pollingStations`:
```json
{
    "id": "PS006",
    "name": "New Station",
    "location": "New Location",
    "capacity": 500,
    "status": "Ready"
}
```
